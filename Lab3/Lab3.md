## Самостоятельная работа №5
### Создать программу, которая будет выводить в консоль 'hello world', под ним 'hello' 5 раз и закончит работать предложением 'hello world'

```python
string = 'hello'
values = [0, 2, 4, 6, 8, 10]
counter = 0
while ' world' not in string:
    memory = string
    if counter in values:
        string = string + ' world'
    print(string)
    if counter < 10:
        string = memory
    counter += 1
```

values = [0, 2, 4, 6, 8, 10]

Дальше идет проверка, что пока счетчик меньше переменная 'memory' вновь переписвается и счетчик в конце концов повышается на 1 значение. И так пока счетчик не будет равен 10.
